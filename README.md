🏋️ Rep Counter

A simple and responsive Rep Counter web application built using HTML, CSS, and JavaScript. This project helps users count workout repetitions and save previous rep counts with a clean fitness-themed interface.

✨ Features
➕ Increment rep count
💾 Save previous rep entries
🔄 Reset count after saving
🎨 Modern fitness-themed UI
📱 Responsive design
⚡ Fast and lightweight (No frameworks)
🛠️ Technologies Used
HTML5
CSS3
JavaScript
